# Getting Started with the Web Storage API in Vanilla JavaScript  
## Building a Girls Act Register – Beginner Toolkit

---

## 1. Title & Objective

**Chosen Technology:** Web Storage API (`localStorage`) with Vanilla JavaScript  

**Why I chose it:**  
I wanted to learn how to build a simple web application that works directly in the browser without needing a backend or database at the start. `localStorage` is beginner-friendly and allows storing data persistently on the user's browser.

**End Goal:**  
To build a **Girls Act Participant Register** that:
- Collects participant data using a form  
- Stores data in the browser  
- Displays saved entries  
- Exports data as CSV or JSON  

---

## 2. Quick Summary of the Technology

The **Web Storage API** allows web applications to store data in the browser.

### Key Feature:
- `localStorage` stores data **permanently** (until manually cleared)

### Where it is used:
- Saving user preferences  
- Offline apps  
- Small data storage without a database  

### Real-world Example:
A website remembering your login session or theme (dark/light mode).

---
