AHF Girls Act Register

This starter register is designed for programme intake, attendance, and follow-up tracking for Girls Act activities.

Important: this demo stores submissions only in the browser on the current device using local storage. That is fine for prototyping, training, and non-sensitive testing, but it is not the right final setup for sensitive or health-related participant data. Use GitHub Pages for the interface only, and connect it to a secure backend later.
